Login App

A simple iOS login and registration application built using Swift and UIKit. This project provides a modern user interface for logging in and signing up, and demonstrates usage of features like UITextField customization, Auto Layout, and UIView animations. The app uses a series of well-constructed view controllers for different screens, such as login, registration, and onboarding.

Features

Login and Registration Screens: Allows users to register with an email and password, and log in using the same credentials.

Custom Floating Label Text Fields: Beautiful text fields with floating labels and bottom borders for an engaging user experience.

Onboarding Screen: An introductory screen with a descriptive title and interactive buttons for Login and Register.

Navigation Between Screens: Smooth navigation between Login, Register, and other views using Navigation Controllers.

User-Friendly Design: Includes vibrant colors, attractive icons, and intuitive UI elements.

Dynamic Animations: Animated elements, such as button clicks and transitions.

High Scores Feature: Displays player high scores in the game section.

Installation

Clone the Repository

git clone https://github.com/shaileshsrigiri/Login-App.git

Open in Xcode

Open the Login-App.xcodeproj file using Xcode.

Run the Application

Select a target simulator and press the Run button or Cmd + R.

How to Use

Onboarding

Launch the app to view an introductory screen that provides options to log in or register.

Login Screen

Enter your registered email and password to log in.

Register Screen

Register as a new user with your name, email, and password.

Customization

You can modify the text fields, buttons, and other UI elements by changing their properties within the LoginViewController.swift or RegisterViewController.swift files.

The high scores functionality is built for demonstration; it can be further expanded for full backend integration.
